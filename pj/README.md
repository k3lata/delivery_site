# 🍽️ Food Delivery Web Application

## 📌 Описание проекта

Данный проект представляет собой веб-приложение для онлайн-доставки еды и составления меню.

Пользователь может:
- просматривать блюда по кухням
- добавлять блюда в корзину
- оформлять заказ
- сохранять адреса и банковские карты
- формировать персональное меню (Menu Builder)
- управлять своим профилем

Администратор может:
- просматривать все заказы
- изменять статус заказов

---

## ⚙️ Используемые технологии

- React
- React Router
- Context API

### Backend:

- Express
- MongoDB (Mongoose)
- JWT (аутентификация)

- multer (загрузка файлов)

## 🚀 Запуск проекта

### 1. Клонирование проекта

```bash
git clone <ссылка на репозиторий>
cd project
2. Запуск сервера
cd server
npm install
npm start

Сервер запустится на:

http://localhost:5000
3. Запуск клиента
cd client
npm install
npm run dev

Клиент будет доступен по адресу:

http://localhost:5173
